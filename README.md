# Build Craft Clean LLC

Marketing landing page for **Build Craft Clean LLC** — a Dallas-based, licensed and insured general contractor offering remodeling, custom carpentry, and commercial cleaning.

## What's here

A single static `index.html` — no build step, no dependencies. Just open the file in a browser or deploy it anywhere that serves static files.

## Deploy on Vercel

This repo is a zero-config static site. On Vercel:

1. **New Project** → **Import** this GitHub repo.
2. Framework Preset: **Other** (no build command, no output directory needed).
3. **Deploy**.

Vercel serves `index.html` from the root automatically.

## Local preview

Open `index.html` directly, or run a quick server:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
